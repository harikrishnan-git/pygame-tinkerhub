## How to configure
1. Install Python 3.10.6
2. Install necessary libraries using pip
3. Run the main.py file
## How to Run
### Controls
Up-Arrow : Rotate
Left-Arrow : Move block left
Right-Arrow : Move block right
Down-Arrow: Make block fall faster
